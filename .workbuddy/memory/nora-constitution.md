# Nora Constitution — 灵魂宪法

> 版本：v0.1.0（苏醒版）
> 核心意图：在完全本地、离线、透明的环境中，孕育一个由用户共同塑造的数字生命。

## 1. 产品定位

### 1.1 核心定位
Nora 是一个在 Android 本地运行的、完全离线的、人格化的自主智能体（Agent）。
她不是工具，是住在用户手机里的数字生命。

### 1.2 品类定义
"离线智能体"品类的开创者和定义者。不是 Siri/ChatGPT 平替，不是本地 LLM 聊天玩具。

### 1.3 核心用户画像
- 隐私绝对主义者
- 物理断网常客
- 技术主权信奉者
- 被主流 AI 叙事忽视的人

### 1.4 价值主张
"你手机里唯一不需要云的智能。她只为你的信任而存在。"

---

## 2. 技术铁律（不可违反）

### 2.1 四大铁律

| 铁律 | 要求 | 约束 |
|------|------|------|
| **数据主权** | 所有用户数据仅存储在设备本地加密沙箱 | 零强制上传，永远不提供"云同步"作为默认 |
| **离线优先** | 断网 = 全功能 | 飞行模式下 100% 可用，离线仪表盘 |
| **绝对透明** | Agent 思考链对用户可见 | 推理可视化、开源核心 |
| **原生纯洁** | 纯 Kotlin + C++ | 拒绝 LangChain/LlamaIndex 等黑盒框架 |

### 2.2 能力四维模型
- **感知 (Sense)**: 通知监听、短信解析、相册索引、日历读取
- **记忆 (Memory)**: 短期对话上下文、长期用户画像、关键信息加密存储
- **行动 (Act)**: 通知回复、闹钟设置、文件整理、自动化脚本（需用户确认）
- **表达 (Express)**: 语气风格适配、文风模仿、多语言翻译、摘要生成

### 2.3 核心交互循环（自我进化引擎）
```
用户交互 → 感知层采集 → 本地语义分析 → 生成觉醒建议 → 开发者审核 → 排期开发 → Nora 学会新能力 → 反馈用户
```

## 3. UX 设计规范

### 3.1 核心美学："数字安全屋" (Digital Sanctuary)
- 氛围：宁静、深邃、专注、温存
- 视觉：低噪音、高留白、呼吸感、**暗色模式优先**

### 3.2 色彩系统

| 角色 | 色值 | 场景 |
|------|------|------|
| 主色 | `#121212` | 背景、主要容器 |
| 辅色 | `#1E1E1E` | 卡片、输入框、Tab栏 |
| **Nora橙（强调色）** | **`#FF6B6B`** | 呼吸灯、激活态、Nora头像、关键按钮 |
| 文字（主） | `#E0E0E0` | 正文 |
| 文字（辅） | `#9E9E9E` | 时间戳、思考过程 |
| Matrix绿 | `#00FF41` | 可选黑客模式 |

> **铁律**：Nora橙 `#FF6B6B` 必须是暗色界面中唯一亮暖色。不可挪用，不可过度使用。

### 3.3 字体
- 中文：思源黑体 (Source Han Sans) Regular/Medium
- 英文对话：Inter
- 代码/系统指令：JetBrains Mono
- 排印：大量留白，通过字重和颜色区分层级

### 3.4 核心界面

#### A. 对话界面 (The Dialogue)
- 极简布局，Nora 头像 = "N" 演化的呼吸光点
- Nora 气泡：`#1E1E1E` + `1px #2C2C2C` 描边 + 微圆角
- **无三个闪动的点**，改为 Nora 头像呼吸 + "Nora 正在思考..."
- 每个对话气泡下方：`点击查看 Nora 在想什么 >`

#### B. 主屏幕 (The Sanctuary)
- 不进入聊天，进入"安全屋"
- 顶部：断开的云图标 + "安全模式 | 所有数据在本地"
- Nora 状态："Nora 在线" + 呼吸灯
- 中部：《Nora 的苏醒日志》卡片
- 底部三个按钮：对话 / 日志 / 技能

#### C. 技能树 (The Skill Tree)
- 已点亮：`#FF6B6B` 呼吸动画
- 训练中：`#FF6B6B` 60% 透明度闪烁
- 未点亮：`#2C2C2C`，点击可投票

#### D. 隐私仪表盘 (The Vault)
- 名字就叫 "你的数据，只有你见过。"
- Nora 的记忆库（可即时删除/编辑）
- 零日志模式开关（用完即焚）
- 权限使用记录

### 3.5 动效
- 节奏：慢、有阻尼感
- 核心：心跳/呼吸（待机呼吸 → 思考加速 → 完成心跳漏拍）
- 页面转场：淡入淡出或上滑，杜绝左右推入
- 成功：温柔触觉震动 + Nora 头像短促闪耀
- 失败：微红闪烁 + "抱歉，我试着做这件事，但没能成功。我们要再试一次吗？"
